Smart Face Attendance System

Project Description

A modern, AI-powered attendance management system that uses facial recognition technology to mark student attendance automatically. Built with Python Flask, OpenCV, HTML, CSS.

✨ Key Features

  • Real-time Face Recognition: Uses LBPH algorithm for accurate face detection and identification from webcam feed.
  • Smart Attendance Rules: Includes a 1-hour cooldown policy to prevent duplicate entries for the same student.
  • Automated Model Training: The recognition model retrains automatically whenever a new student is registered.
  • Admin-Protected Registration: Student registration is password-protected (default: admin123) for secure access.
  • Export to Excel: All attendance records are automatically saved in Excel format for easy reporting and analysis.
  • Modern Glass UI: Features a professional user interface with glass morphism design, responsive layouts, and smooth animations.

Technologies Used

Python Flask OpenCV Pandas HTML CSS JavaScript

How It Works

  1. Register: Admin adds new students with face samples (100 images captured via webcam).
  2. Train: System automatically trains the model with new face data.
  3. Mark: Student looks at camera to mark attendance with real-time verification.
  4. Track: Records are saved with timestamps, preventing duplicate entries within 1 hour.
